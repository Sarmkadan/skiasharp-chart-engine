# PerformanceOptimizer

The `PerformanceOptimizer` class provides comprehensive diagnostic tools for evaluating and optimizing the performance of charts within the `skiasharp-chart-engine`. It facilitates performance analysis, data downsampling to reduce rendering overhead, and memory footprint estimation, while offering actionable recommendations to improve chart responsiveness and resource utilization.

## API

### Constructors

- **`PerformanceOptimizer()`**
  Initializes a new instance of the `PerformanceOptimizer` class.

### Methods

- **`PerformanceAnalysis AnalyzeChart(Chart chart)`**
  Performs a comprehensive performance analysis on the provided chart instance, identifying potential bottlenecks in rendering or data processing.
  - **Returns**: A `PerformanceAnalysis` object containing the results of the evaluation.

- **`List<DataPoint> Downsample(List<DataPoint> data, int targetCount)`**
  Reduces the number of data points in the provided list to the specified `targetCount` while attempting to preserve the visual characteristics and trends of the original dataset.
  - **Parameters**: 
    - `data`: The original list of `DataPoint` objects.
    - `targetCount`: The desired maximum number of data points.
  - **Returns**: A new `List<DataPoint>` representing the downsampled data.

- **`long EstimateMemoryUsage(IEnumerable<DataPoint> data)`**
  Calculates an estimate of the memory footprint required to process and render the provided dataset.
  - **Parameters**: 
    - `data`: The collection of `DataPoint` objects to analyze.
  - **Returns**: A `long` value representing the estimated memory usage in bytes.

### Properties

- **`string ChartId`**
  Gets or sets the unique identifier associated with the chart being optimized.

- **`int TotalDataPoints`**
  Gets or sets the total number of data points currently held or analyzed by the optimizer.

- **`DateTime AnalyzedAt`**
  Gets or sets the timestamp indicating when the last performance analysis was performed.

- **`List<OptimizationRecommendation> Recommendations`**
  Gets or sets the collection of recommendations generated by the most recent performance analysis.

- **`string Category`**
  Gets or sets the categorization of the performance issue or recommendation (e.g., "Rendering", "Memory").

- **`Severity Severity`**
  Gets or sets the `Severity` level (e.g., Low, Medium, High, Critical) of the identified issue or recommendation.

- **`string Message`**
  Gets or sets the detailed descriptive message regarding the performance finding.

- **`string Action`**
  Gets or sets the recommended corrective action to address the identified performance concern.

## Usage

### Example 1: Analyzing and Optimizing a Chart
```csharp
var optimizer = new PerformanceOptimizer { ChartId = "RevenueChart_001" };
var analysis = optimizer.AnalyzeChart(myChart);

if (optimizer.Recommendations.Any(r => r.Severity == Severity.Critical))
{
    Console.WriteLine($"Critical issue found: {optimizer.Message}");
    Console.WriteLine($"Action: {optimizer.Action}");
}
```

### Example 2: Downsampling Data for Rendering
```csharp
var rawData = GetDataFromSource(); // Assume this returns a very large list
var optimizer = new PerformanceOptimizer();

// Reduce data points to 1000 for efficient rendering
var optimizedData = optimizer.Downsample(rawData, 1000);
myChart.DataSource = optimizedData;
```

## Notes

- **Thread Safety**: This class is not inherently thread-safe. Instances should not be shared across multiple threads without external synchronization if properties like `ChartId`, `Category`, or `Recommendations` are being modified concurrently.
- **Data Integrity**: The `Downsample` method is intended for visual representation. Do not use downsampled data if precise calculation of aggregate statistics (such as exact mean or sum) is required, as data point loss is expected during the reduction process.
- **Memory Estimation**: `EstimateMemoryUsage` provides an approximation based on the average size of `DataPoint` objects and engine overhead; actual runtime memory consumption may vary depending on the garbage collection state and runtime environment.
